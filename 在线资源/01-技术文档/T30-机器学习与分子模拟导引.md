分子动力学产生**结构与轨迹**，机器学习则从数据中学习规律、做预测或拟合势能面。二者结合已是当前计算材料、化学模拟的重要方向，但文献庞杂，初学者易与「通用 AI 课」或「ChatGPT 写脚本」混为一谈。

本文是 MolSimulX **ML / AI 的技术落地导引**（环境、路线图、示例索引）。**概念原理**请先读 [机器学习数据基础](K28-机器学习数据基础.md)、[神经网络与深度学习基础](K29-神经网络与深度学习基础.md)。代码入门见 [NumPy与Matplotlib简明教程](C01-NumPy与Matplotlib简明教程.md)（免费）、[MDAnalysis](C02-MDAnalysis轨迹分析入门.md)、[ASE](C03-ASE结构构建入门.md)（部分 VIP，「一、」可免费读）。

## 一、MolSimulX 在 ML 方向做什么、不做什么

| 我们做 | 我们不主打 |
|--------|------------|
| 用 **MD 轨迹 / 结构** 做统计学习与性质预测 | 通用计算机视觉、NLP 课程 |

| **ML 势** 与 LAMMPS 联用入门 | 大规模 MLOps / K8s |

| **Cursor** 辅助写分析脚本（见 VSCode 教程） | 替代第一性原理或实验 |

## 四、环境配置（重要）

MolSimulX 建议 **分层环境**，避免单个 env 过大、CUDA 与 CPU 机器冲突。

### 1. `myenv` —— MD 分析 + 轻量 ML（已有）

[分子模拟工作平台搭建](T01-分子模拟工作平台搭建.md) 中的一键包已含：

- MDAnalysis、ASE、NumPy、Pandas、Matplotlib、JupyterLab  
- **足够完成**：轨迹分析、描述符、**scikit-learn** 级统计学习  

若已部署 `myenv`，只需追加 sklearn（若未装）：

```bash
conda activate myenv
mamba install scikit-learn -y
```

### 2. `myenv-ml` —— 机器学习扩展（推荐新建）

在 `codes/myenv-ml.yml` 中预置：

| 组件 | 用途 |
|------|------|
| Python 3.12 | 与 myenv 一致 |
| MDAnalysis、ASE | 与 MD 无缝衔接 |
| **scikit-learn** | 回归、分类、聚类、降维 |
| **PyTorch** (`pip`) | 神经网络、GNN、ML 势训练 |
| **RDKit** (`pip`) | 分子描述符、SMILES |
| JupyterLab | Notebook |

**创建环境：**

```bash
cd /path/to/MolSimulX网站/codes
mamba env create -f myenv-ml.yml
conda activate myenv-ml
python -m ipykernel install --user --name myenv-ml --display-name "Python (myenv-ml)"
```

在 VSCode / Jupyter 中选 Kernel **Python (myenv-ml)**。

> **Tips：** `torch` 默认 pip 安装多为 CPU 版。若需 **GPU**，请到 [PyTorch 官网](https://pytorch.org/) 按 CUDA 版本选择安装命令，在已激活的 `myenv-ml` 中**重装** torch，并记录在项目 README。

### 3. `myenv-mlip` —— 机器学习势（进阶，后续文档）

专用于 MACE、NequIP 等框架，依赖重、常需 GPU。将在「ML 势」系列教程中给出**独立 yml**，不与 `myenv-ml` 混装。

### 4. 环境选择速查

| 任务 | 环境 |
|------|------|
| RDF、RMSD、画图 | `myenv` |
| 聚类、随机森林、PyTorch 小实验 | `myenv-ml` |
| 训练 ML 势、大规模 GNN | `myenv-ml` 或 `myenv-mlip` + **集群 GPU** |
| 仅编辑脚本 | 本地；训练提交集群（[集群与SLURM简明教程](T10-集群与SLURM简明教程.md)） |

### 5. 项目目录建议

在 [科研项目目录结构规范](T15-科研项目目录结构规范.md) 基础上增加：

```
my-project/
├── models/              # 保存 .pkl / .pt 权重
├── data/
│   ├── raw/             # 轨迹
│   ├── processed/       # 特征表 csv、numpy
│   └── labels/          # 物性标签、QM 结果
├── notebooks/ml/        # ML 实验 Notebook
└── environment.yml      # 或注明 myenv-ml
```

## 八、数据与科研伦理（简记）

- **训练 / 测试划分**：按**分子或独立模拟**划分，勿将同一轨迹相邻帧同时放进训练与测试（信息泄漏）。  
- **标签来源**：实验 / DFT / 经验势须在 Methods 中说明。  
- **未发表数据**：上传云端训练或 Cursor AI 时注意组内与平台政策。  
- **开源协议**：QM9、MD17 等数据集各有许可，商用前核对。

## 十、小结

1. MolSimulX ML 线 **锚定在 MD 数据与 LAMMPS 工作流**，不是通用 AI 课。  
2. **`myenv`** 做分析 + 轻量 ML；**`myenv-ml`** 加 sklearn、PyTorch、RDKit。  
3. **MDAnalysis / ASE** 是数据入口；后续按本路线图逐篇补教程。  
4. 案例与 ML 势深度内容放在 **`实战案例`** 与进阶 env。
