> **系列标签：** `实战案例` · `栏目说明`

本栏目是 **分子模拟相关的动手案例**：围绕具体体系或工作流（建模 → 模拟 → 分析 → 出图，或轨迹 ML、ML 势验证等），把知识与工具串成一条可复现路径。

**不放这里：** NumPy / Matplotlib、sklearn、PyTorch 等**工具速成**——已规划归入 [技术文档](../01-技术文档/)（见写作手册「工具速成」；现稿暂仍以 C01–C05 文件名放在本目录，迁入前可继续点开阅读）。

**访问：** 站点分类选 **实战案例**。

| 标签 | YAML | 含义 |
|------|------|------|
| **免费** | `free` | 全文公开（引流案例） |
| **VIP** | `vip` | 全文会员可读 |
| **VIP下载** / **付费下载** | `download-vip` / `download-paid` | 正文免费，转化在资源包（端到端案例常用） |

环境请先按 [分子模拟工作平台搭建](../01-技术文档/T01-分子模拟工作平台搭建.md) 配置 `myenv`；做 ML 案例时再用 `mymlenv`（见 [机器学习内容规划](../../机器学习内容规划.md)）。
