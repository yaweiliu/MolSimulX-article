## 一、粗粒化之后，「时间」常常变快了

粗粒（CG）盒子里，珠子扩散、相分离、自组装，经常比同体系的全原子（AA）轨迹**在墙钟时间上更快走完类似过程**。原因通常叠在一起：

| 因素 | 在干什么 | 对动力学的影响 |
|------|----------|----------------|
| **粒子变少** | 非键对数下降，每步更便宜 | 同样算力能推进更长的「模型时间」 |
| **势能更平滑** | 抹掉快振动、细沟槽后，有效势往往更「软」 | 跨垒更容易；有时还能用更大 $\Delta t$ |
| **缺了隐式摩擦** | 显式溶剂、键振动等会不断碰撞 / 拖曳 | 去掉后，有效摩擦常变小 → 扩散变快 |
| **映射本身** | 一珠代表一团原子的质心运动 | 观测量定义在粗坐标上，时间尺度不必等于真实 ns |

口诀：**更快 ≠ 算错了**，而是模型自由度变了。比「谁先分相、谁扩散更快」这类**相对**结论往往仍有用；把 CG 的 1 ns 直接当成实验 1 ns，则要非常小心。

> **Tips：** Methods 里若报扩散系数、弛豫时间，写清：粗粒化映射、是否另加耗散 / 朗之万摩擦、以及时间是否做过标定（相对 AA 或实验）。

## 二、从熵看：抹掉自由度，留下「有效势」

统计力学图像（细节见 [统计力学基础与系综](K23-统计力学基础与系综.md)）：全原子构型空间极大。粗粒化相当于对一部分坐标做**积分掉 / 平均掉**（溶剂内部、快振动、局部取向……），在留下的粗坐标上得到**有效相互作用**。

这条有效相互作用，严格说往往不是「真空里的势能 $U$」，而是带温度的**有效自由能 / 势平均值**（potential of mean force 一类图像）：

- 被抹掉的那些自由度，贡献的不只是能量，还有**熵**——构型越多、越「乱得有空间」，有效自由能里就越有一项 $-TS$ 味道的贡献；  
- 结果是：粗粒势能面常比全原子面**更平滑、垒更矮或更宽**，粒子在粗坐标上「滑」得更快；  
- 同时，被积掉的快自由度本来也在提供**耗散与噪声**（溶剂碰撞就是典型）。粗粒方程若只留下保守力 $\mathbf{F}=-\nabla U_{\mathrm{eff}}$，等于把摩擦也一并抹淡了——动力学再被加速一截。

所以「CG 动力学偏快」里，至少有两层：

1. **热力学层**：有效自由能景观更易探索（熵 / 平均势抹平细节）；  
2. **动力学层**：缺了溶剂与内部模式带来的摩擦与随机力。

只补保守有效势、不补耗散，对**平衡分布**有时仍可调参对准；对**时间相关量**（扩散、粘度、流变）则往往系统偏快。

## 三、对寻找平衡态：多数时候是好事

本站主线是**经典 MD**，多数课题、也是入门起点，落在**平衡态**一侧（见 [分子动力学模拟概述](K02-分子动力学模拟概述.md)）：在设定系综下采稳得住的结构 / 热力学量。非平衡（流变、驱动测输运等）同样在地图上，只是通常后学。

粗粒动力学偏快，对找平衡态常是**加分项**：

| 你想要什么 | CG 偏快帮你什么 |
|------------|-----------------|
| 膜成囊泡、相分离、聚合物分相 | 更快扫过构型空间，更容易看见「最终像什么样」 |
| 多起步、多温度扫趋势 | 同样算力下样本更多，平衡化更省 |
| 先看有没有稳态 / 两相 | 墙钟时间里更快到平台，再决定要不要回全原子细校 |

注意边界：

- **热力学要对**：有效势 / 参数若偏了，你只是更快地采到**错误**平衡态；  
- **观测量要在粗分辨率下仍有意义**：问原子级氢键取向，CG 再快也答不对；  
- **「平衡化够了」仍要判据**：平台、分段一致，见 [平衡判据与收敛](K13-平衡判据与收敛.md)——快不等于免检。

经验工作流仍是：[粗粒化与加速模型](K04-粗粒化与加速模型.md) 里说的「粗扫趋势 → 细校关键量」。

## 四、非平衡与流变：偏快反而可能坏事

非平衡过程关心的是**响应怎么随时间建立、通量多大**——例如剪切下的应力、粘度、松弛谱（见 [输运系数谱系](K21-输运系数谱系.md)、[非平衡分子动力学概述](K22-非平衡分子动力学概述.md)）。

这时「CG 时间跑得太欢」会直接扭曲结论：

| 现象 | 风险 |
|------|------|
| 有效摩擦过小 | 扩散、应力松弛过快；表观粘度偏低 |
| 只用保守粗粒势做剪切 | 缺水动力学 / 缺耗散，流变与实验对不上数量级 |
| 把 CG 时间轴当真实时间 | 应变速率与弛豫时间的**无量纲组合**全错（见第五节） |

**可以适当增加耗散，把动力学「拉慢」回来**——不是推翻粗粒化，而是把第二节里被抹掉的摩擦 / 噪声，用有效方式加回去：

| 做法 | 在补什么 | 去向 |
|------|----------|------|
| **朗之万摩擦 + 随机力** | 单粒子级耗散与热噪声 | [朗之万、布朗与溶剂介质方法](K25-朗之万布朗与溶剂介质方法.md) |
| **DPD 成对耗散 / 随机** | 动量更友好的介观耗散 | 同左；亦见 [粗粒化与加速模型](K04-粗粒化与加速模型.md) |
| **相对 AA / 实验标定 $\gamma$ 或时间因子** | 把模型时间映回物理时间 | Methods 写清标定对象与适用范围 |
| **关键步骤回全原子短跑** | 用细模型钉住局部机理或系数 | 多尺度细校 |

原则：**平衡结构 / 相行为**可以享受 CG 加速；**流变、输运系数的绝对值、与真实应变速率对齐的非平衡过程**，要显式处理耗散与时间尺度，并核对下面的无量纲数——不能默认「并珠子就完事」。

> **Tips：** 加耗散会改动力学，一般**不应**指望它自动修正错误的有效保守势。先让平衡性质大致靠谱，再调摩擦去对扩散或粘度。

## 五、无量纲数：比的是「谁强谁弱」

对比单位把长度、能量换成 $^*$（见 [对比单位与无量纲化](K15-对比单位与无量纲化.md)），是换尺子。这里说的**无量纲数**另一层意思更贴非平衡：把**两种效应的特征尺度之比**写成一个纯数，用来判断「现在主导的是哪一边」。

典型形状是：

$$
\text{无量纲数} \sim \frac{\text{过程 A 的特征速率（或力、通量）}}{\text{过程 B 的特征速率（或力、通量）}}
$$

数量级对了，物理图像才对；差一个数量级，往往不是「差一点」，而是**换了机制**。

### 1. 流变与驱动里常见的几个

不必背全表；先记住「它在比什么」：

| 常见名字 | 大致在比什么 | 为什么在乎数量级 |
|----------|--------------|------------------|
| **Weissenberg** $\mathrm{Wi}\sim\dot\gamma\,\tau$ | 剪切有多快 vs 应力 / 构象松弛有多快 | $\mathrm{Wi}\ll 1$ 近线性；大了进入非线性流变 |
| **Deborah** $\mathrm{De}\sim\tau/t_{\mathrm{obs}}$ | 内禀弛豫 vs 你观察 / 加载的时间 | 短时间像固体、长时间像液体的分界 |
| **Péclet** $\mathrm{Pe}\sim$（对流或漂移）/ 扩散 | 外场拖着走 vs 布朗扩散抹平 | 决定结构是否被剪切「拉开」还是仍扩散主导 |
| **雷诺** $\mathrm{Re}$（介观流体时） | 惯性 vs 粘滞 | 分子 / 胶体盒子多数 $\mathrm{Re}$ 很小；别误进惯性主导区 |

名称会随领域变写法；Methods 里写清定义（用的是哪个 $\tau$、哪个特征长度）比纠结中译更重要。

### 2. 和「CG 偏快」怎么拧在一起

粗粒化常把扩散变快、把有效松弛时间 $\tau$ 变短。若你仍按实验或全原子的物理 $\dot\gamma$ 去设剪切，盒子里的 $\mathrm{Wi}=\dot\gamma\,\tau$ **会系统偏小**——你以为在探非线性，其实还在线性尾巴上；反过来，若把 CG 时间轴误当成真实时间去报 $\dot\gamma$，无量纲数又可能虚高。

因此，非平衡粗粒化里，调耗散 / 标定时间，目标常常不是「让 ns 数字好看」，而是：

1. 先定你要对齐的**目标无量纲数**（例如实验或介观理论所在的 $\mathrm{Wi}$、$\mathrm{Pe}$ 量级）；  
2. 再据此选 $\dot\gamma$、摩擦 $\gamma$、或时间映射因子，使盒子里的比值落在同一数量级；  
3. 扫一两个相邻量级，确认结论对「谁主导」不敏感，或明确写出已进入非线性。

对许多非平衡过程，**先保证无量纲数数量级对，比抠某一个绝对系数更值钱**——绝对粘度可以后面再细校；机制错了，细校也是在错区里打转。

### 3. 实践上盯什么

| 检查 | 问自己 |
|------|--------|
| 定义 | $\tau$、特征长度、特征速度是从轨迹哪一段、哪个观测量来的？ |
| 目标 | 要对齐实验 / 理论的哪个无量纲窗口，还是只要定性趋势？ |
| CG 影响 | 并珠、加摩擦之后，$\tau$ 与扩散变了多少？无量纲数跟着漂了吗？ |
| 线性区 | NEMD 外推前，$\mathrm{Wi}$ 或驱动强度是否仍落在线性区？（见 [NEMD](K22-非平衡分子动力学概述.md)） |
| 报道 | 写出无量纲数的定义与取值范围，而不仅是「剪切速率 $=\,x\,\mathrm{s}^{-1}$」 |

> **Tips：** 平衡态扫相图，无量纲状态点（如 $T^*$、$\rho^*$）已经在管「处在哪」；非平衡再多一层——无量纲**驱动强度**管「推得有多狠相对内禀动力学」。两层都对，跨模型对比才公平。

## 六、一张对照表

| | **平衡态为主** | **非平衡 / 流变为主** |
|--|----------------|------------------------|
| CG 动力学偏快 | 常有利：更快找平台、扫趋势 | 常有害：时间轴与耗散不对 |
| 熵 / 有效自由能 | 平滑景观 → 更好探索 | 仍要热力学对；另补耗散管时间 |
| 无量纲数 | 状态点（$T^*$、$\rho^*$ 等）对齐即可 | **驱动 vs 弛豫 / 扩散**的比值数量级要对 |
| 优先动作 | 检查映射与有效势；粗扫 + 细校 | 加摩擦或标定时间；对准 $\mathrm{Wi}$/$\mathrm{Pe}$ 等；对照 [NEMD](K22-非平衡分子动力学概述.md) |
| 报道时 | 相对趋势、结构、相图 | 写清耗散、时间映射、无量纲数定义与线性区 |

## 七、小结

1. 粗粒化后动力学常变快：粒子更少、势更平滑、缺了溶剂 / 快模式带来的摩擦。  
2. **熵角度**：积掉自由度 → 有效自由能景观更易探索；同时耗散通道被削弱。  
3. 对**寻找平衡态**，偏快多半是好事（仍要模型热力学靠谱、观测量匹配分辨率）。  
4. 对**流变等非平衡过程**，偏快会扭曲时间相关量；可**增加耗散项**（朗之万、DPD 等）减慢动力学，或标定时间因子。  
5. **无量纲数**刻画两种效应谁强谁弱；CG 改了 $\tau$ 与扩散后，要重算并对齐数量级——对许多非平衡过程，这比单独抠绝对速率更关键。  
6. 保守有效势管「停在哪」，耗散管「走多快」，无量纲数管「推得有多狠相对内禀动力学」——分开调，Methods 分开写。
