## 一、从线性模型到神经网络

**线性回归：** $y = w x + b$，只能拟合直线。

**多层感知机（MLP）：** 多层「神经元」与非线性激活（ReLU、tanh）叠加，可逼近复杂函数：

```
输入（如原子间距 r）
    → 隐藏层 1 + 激活
    → 隐藏层 2 + 激活
    → 输出（如能量 U）
```

**深度学习** = 层数/参数量较大的神经网络，用 **反向传播** 和 **梯度下降** 训练。

---

## 二、核心术语

| 术语 | 含义 |
|------|------|
| **参数 / 权重** | 模型学习得到的 $w$、$b$ 及层间矩阵 |
| **前向传播** | 输入 → 逐层计算 → 预测输出 |
| **损失函数** | 预测与真值的差距（MSE、交叉熵等） |
| **反向传播** | 求损失对各参数的梯度 |
| **优化器** | Adam、SGD 等，用梯度更新参数 |
| **epoch** | 全训练集过一遍 |
| **batch** | 每次更新用的一小批样本 |
| **过拟合** | 训练好、测试差；样本少或模型过大 |
| **泛化** | 在新数据上仍准确 |

---

## 三、训练在做什么？

最小化损失：

$$
\min_{\theta} \frac{1}{N} \sum_i L(f_\theta(x_i), y_i)
$$

循环：**取 batch → 前向 → 算 loss → backward → 更新 $\theta$**（见 PyTorch 教程 LJ 拟合例）。

**超参数**（学习率、层宽、层数）常在验证集上调，不用测试集。

## 四、与 sklearn 的分工

| sklearn | 神经网络 / PyTorch |
|---------|-------------------|
| 随机森林、SVM 等 | 自定义网络结构 |
| `.fit()` 一行 | 自写训练循环 |
| 表格特征 | 图、序列、三维场 |
| 中小数据快速基线 | 大数据、端到端表示学习 |

许多项目：**sklearn 基线 → 再上神经网络**。

## 五、在分子模拟中的三类应用

### 1. 后处理：轨迹 / 描述符 → 性质

输入：RDF、序参量、指纹  
输出：相态、扩散系数等  
网络规模小，CPU 可训。

### 2. 结构–性质：分子图 → 标量

**图神经网络（GNN）** 把原子当节点、化学键当边，消息传递聚合邻域信息，预测能量、毒性等（进阶教程待写）。

### 3. 机器学习势（MLIP）

输入：每个原子的**局部环境**（邻居种类、距离、角度）  
输出：体系**总能量**；对坐标求导得**力**  

力驱动 MD，替代或补充 [经典全原子力场](K03-经典全原子力场.md) 中的固定函数形式。

| 代表思路 | 说明 |
|----------|------|
| Behler–Parrinello | 对称函数描述环境 + 神经网络 |
| SchNet、MACE、NequIP 等 | 等变 GNN，尊重旋转平移对称性 |
| 训练数据 | 多为 DFT 计算的 E、F |

**外推风险：** 离开训练化学空间，能量/力可能不可靠。

## 六、对称性为何重要

物理能量在旋转、平移下不变；某些表示在交换等价原子上不变。  
**等变神经网络** 把对称性编入架构，比「裸 MLP + 坐标」样本效率高、更物理。

初学者只需知道：**ML 势论文常强调 E(3) 等变**；不必先手推公式。

## 八、与「人工智能」大模型

| 类型 | 与 MD 关系 |
|------|------------|
| **ChatGPT / Cursor 等 LLM** | 辅助写脚本、解释报错；**不**替代力场或积分 |
