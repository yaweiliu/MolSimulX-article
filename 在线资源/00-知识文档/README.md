
> **系列标签：** `知识文档` · `栏目说明` · `学习路径` · `MolSimulX`

本栏目讲**概念和原理**——按「做一次经典 MD」的流程组织：方法地图 → MD 总览 → 力场 → 建盒子与算力 → 积分与控温 → 平衡与分析（含输运 / NEMD）→ 统计力学加深 → 其他方法 / ML。**不涉及装软件、敲命令**。适合开跑模拟前先建立图景，或写论文时回头查定义。

**访问：** 主线概念文（方法 → 开跑 → 平衡 → 基础分析）**免费**；力场两翼、增强采样、分析深化、输运/NEMD、方法边界多为 **VIP**（导语与学习路径可预览）。在站点上按分类 **知识文档** 浏览，或从 [资源导航](../资源导航.md) 进入。划分见写作手册「K01–K27 免费 / VIP」。

文件名带 `K##-` 前缀（与技术文档 `T##-`、实战案例 `C##-` 一致）；正文互链只用**文章标题**，不写编号。

## 和别的栏目怎么配合

```
知识文档  →  搞懂「算什么」
技术文档  →  搭平台、连集群、写笔记
实战案例  →  动手练技能；端到端跑通一整条链路
```

**推荐顺序（零基础，跟一次模拟流程）：**

1. [分子模拟方法概述](K01-分子模拟方法概述.md) → [分子动力学模拟概述](K02-分子动力学模拟概述.md)
2. 力场：[经典全原子力场](K03-经典全原子力场.md) →（按需粗粒化 / 高精度）→ [力场怎么选](K06-力场怎么选.md)
3. 盒子与力：[边界条件与初始条件](K07-边界条件与初始条件.md) → [截断长程力与近邻列表](K08-截断长程力与近邻列表.md)
4. 跑起来：积分 → 约束 → 系综 → 最小化/预平衡 → 收敛
5. 分析：轨迹性质 → 误差 →（按需）尺寸、压强/张力、序参量、[输运](K21-输运系数谱系.md) → [NEMD](K22-非平衡分子动力学概述.md)
6. 加深：[统计力学基础与系综](K23-统计力学基础与系综.md)；按需方法边界、ML
