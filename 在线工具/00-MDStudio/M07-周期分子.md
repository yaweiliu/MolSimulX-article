> **系列标签：** `MDStudio` · `CIF` · `周期分子`

需要一块有晶胞的材料、表面或纳米结构，而不是溶液里漂的孤分子——**周期分子** Tab 用模板帮你长出 `.cif`，再送给力场或装溶剂。

本文扫六类常见模板与输出；CIF 装溶剂的完整配方见 [配方：CIF 框架材料装溶剂 / 客体](M16-配方CIF框架装溶剂.md)。限额与周期性注意点见 [MDStudio 使用须知与限制](M02-MDStudio使用须知与限制.md)。**不讲** 晶体学精修与 DFT 结构优化。

## 一、目标与前提

| 项 | 说明 |
|----|------|
| **目标** | 由模板生成周期性结构 `.cif` |
| **前提** | VIP（常见）；选对「有胞」体系，勿把团簇当晶体 |
| **本 Tab 最易踩** | 生成后忘记：有晶胞材料的力场 / 电荷选项与孤分子不同 |

---

## 二、常见模板类型（短表）

| 类型 | 适用直觉 |
|------|----------|
| 体相晶体 | 金属、简单晶体等 |
| 表面 /  slab | 切面、表面吸附起步 |
| 纳米管 / 线 / 片 | 碳管、纳米片等常见模板 |
| 其他 ASE 模板 | 以界面当前列表为准 |

具体参数（晶向、层数、真空层）按界面填写；生成后在右侧预览看胞是否合理。

---

## 三、操作步骤

1. 打开 **周期分子** Tab。
2. 选择模板类型与参数。
3. 生成并写入工作区。
4. 需要扩胞 / 平移时接 [超胞与几何变换](M10-超胞与几何变换.md)。

---

## 四、产物一览

| 产物 | 用途 |
|------|------|
| `*.cif` | 力场生成、装溶剂/客体、可视化 |

## 六、常见问题

| 问题 | 处理 |
|------|------|
| 打开 CIF 预览怪 | 检查是否选错了模板或真空层过薄 |
| 力场不接受该 CIF | 对照须知：团簇 vs 周期；或改用仓库/自备 CIF |
| 体系太大 | 缩小模板，或先小超胞再扩（注意装盒/冒烟上限） |

---

## 小结

1. 周期分子产出 `.cif`，适合有晶胞体系。
2. 扩胞用超胞 Tab；装溶剂看 CIF 配方篇。
3. 限额与配对注意仍以使用须知为准。
