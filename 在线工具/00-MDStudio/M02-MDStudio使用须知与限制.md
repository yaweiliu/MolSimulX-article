> **系列标签：** `MDStudio` · `限制` · `VIP` · `免费试跑`

MDStudio 能在浏览器里把分子结构变成 Lammps 输入，但它**不是**无限算力的集群节点。用之前先分清两类操作：**轻量级任务**（画、看、管文件、力场转换等）几乎不占共享算力；**重量级任务**（力场生成、装盒、测试模拟、超胞等）会排队、吃墙钟、卡原子数——**平台主要限制的是后者**。

本文写清：哪些算重活、限额数字怎么挡、免费与 VIP 差在哪。逐步怎么点：[Quickstart：从画分子到测试模拟](M01-Quickstart从画分子到测试模拟.md)；界面地图：[MDStudio功能与界面总览](M03-MDStudio功能与界面总览.md)。

## 一、Studio 里的两类任务

下表按**算力体感**划分，**不区分免费 / VIP**（谁能点哪些 Tab，见第二节）。

| 类型      | 典型操作（界面 Tab / 动作）                                                               | 体感                     | 限制侧重点                          |
| ------- | ------------------------------------------------------------------------------- | ---------------------- | ------------------------------ |
| **轻量级** | **孤立分子**、**周期分子**绘制；**分子仓库**浏览 / 预览 / 导入；工作区浏览、重命名、删除、小文件编辑；上传与下载；结构预览；**力场转换** | 秒级交互，一般不占任务队列          | 工作区容量、单文件编辑 / 预览大小、预览结构规模      |
| **重量级** | **力场生成**（含电荷）；**搭建盒子**；**测试模拟**；**超胞变换**                                     | 表单下方出现任务状态、可能排队；数秒～数分钟 | **单分子原子数、装盒总原子、测试原子数、任务墙钟、并发** |

一句话：**轻量级**偏结构编辑、预览、文件与仓库、力场格式转换；**重量级**是力场生成、装盒、测试模拟、超胞。

适合用来装输入与测试，不适合长时间生产 MD、GB 级轨迹在线存、或取代课题组集群。生产轨迹请下载后走本机 / 集群：[分子模拟工作平台搭建](../../在线资源/01-技术文档/T01-分子模拟工作平台搭建.md)、[本地与集群文件传输](../../在线资源/01-技术文档/T09-本地与集群文件传输.md)、[集群与 SLURM 简明教程](../../在线资源/01-技术文档/T10-集群与SLURM简明教程.md)。

当前墙钟与规模是共享机房的保护线；后续会按反馈扩容或对接高性能集群。使用中的需求、限额疑问，或界面 / 任务异常与疑似 bug，欢迎经官网 [联系我们](https://www.molsimulx.com/contact_us/) 反馈；有效 bug 反馈我们会按情况回馈积分。

## 二、账号与权益（谁能点哪些功能）

登录后按会员档位开放 Tab 与额度。**免费**可跟练 Quickstart 主流程里的重量级任务（规模更紧）；更大体系、以及部分轻量级功能（周期分子、力场转换、上传 / 导入等）需 **VIP**。

### 2.1 功能开关

| 能力 | 轻/重 | 免费 | VIP |
|------|-------|------|-----|
| 登录、打开工作室 | — | ✅ | ✅ |
| **孤立分子**绘制；仓库浏览 / 预览 | 轻 | ✅ | ✅ |
| 工作区文件管理（重命名、编辑、删除、下载等） | 轻 | ✅ | ✅ |
| **周期分子**绘制；**力场转换** | 轻 | ❌ | ✅ |
| 仓库**导入**、工作区**上传** | 轻 | ❌ | ✅ |
| **力场生成** | 重 | ✅（电荷见下） | ✅ |
| **搭建盒子**、**测试模拟** | 重 | ✅（规模见第三节） | ✅ |
| **超胞 / 几何变换** | 重 | ❌ | ✅ |

### 2.2 电荷方法（重量级：力场生成）

免费仅可选 **mmff94**、**am1-bcc**。**resp**、**eqeq** 及部分高级选项需 VIP。

### 2.3 免费档怎么用

跟 [Quickstart](M01-Quickstart从画分子到测试模拟.md)：**画乙醇 → 力场 → 装盒约 10 个分子 → 测试模拟**。结构用孤立分子画板生成，不依赖上传或仓库导入。

## 三、重量级任务的规模与时间

限额分两层：**会员档位**（免费更紧）与**机房硬顶**（VIP 接近下表「VIP / 机房」列）。表单、SMILES、保存结构、以及手改过的 `pack.inp`，都会按**当前内容**重算——**改文件名和内容不能绕过**。

### 3.1 规模上限（主要卡重量级）

| 类别 | 主要挡谁 | 免费 | VIP / 机房（典型） | 超出时 |
|------|----------|------|-------------------|--------|
| **单分子原子数** | 力场生成、超胞等；以及绘制 / 保存到工作区过大结构 | **≤ 10** | 约 **1000** | 直接拒绝启动该步 |
| **装盒总原子** | Packmol / 生成盒子 | **≤ 100** | 在线约 **10 万** | 缩小个数；远超时提示本地脚本或线下定制 |
| **测试模拟原子** | 测试模拟 | 受上装盒约束 | 约 **1000** | 无法启动测试 |
| **工作区总容量** | 轻量也会碰 | **50 MB** | 约 **200 MB** | 清理文件 |
| **上传单文件** | VIP 上传（轻量 IO） | 不可上传 | 约 **20 MB** | 失败 |

> **Tips（免费）：** 乙醇约 9 原子 × **10** ≈ 90 原子，落在免费装盒 100 以内。手改 `pack.inp` 的 `number` 再跑 Packmol，会按 `pack.inp` **重算**总原子。

> **Tips（VIP）：** 略超在线上限时，可能给出 `run_packmol.sh` 供本地跑完回传 `simbox.xyz`；**远超 10 万**请 [联系我们](https://www.molsimulx.com/contact_us/)。大分子 AM1-BCC / RESP 也可能「写脚本 + 挂起 → 上传 `.out` 续跑」（需上传权限）。

### 3.2 墙钟与排队（重量级共用）

| 任务（重量级） | 体感 | 超出时 |
|----------------|------|--------|
| 测试模拟 | 往往几十秒～数分钟 | 任务终止，日志提示超时 |
| Packmol / 力场生成 | 可达数分钟 | 同上 |
| 超胞变换 | 视体系大小 | 同上 |

共享机房还有**全局 / 每用户并发**与排队上限：前一个重活没完，下一个会等。具体秒数与队列以界面 / 配置为准。

## 四、安全与习惯

- 勿向公共在线环境上传**未发表、涉密**数据。
- 大轨迹、完整生产输出放本地 / 集群，勿把工作区当硬盘。
- 正式模拟：下载 `data.lmp` / `in.lmp` 后改到本机或集群。

## 五、翻车对照

| 现象                                | 可先检查                                                |
| --------------------------------- | --------------------------------------------------- |
| Tab 提示开通 VIP                      | 周期分子、力场转换、超胞、上传 / 导入需 VIP；孤立分子·力场·装盒·测试可免费用         |
| 电荷没有 resp / eqeq                  | 免费仅 mmff94、am1-bcc                                  |
| 力场 / 导出 / 保存报原子数超限                | 免费单分子 ≤ **10**                                      |
| 装盒报总原子超限                          | 减分子个数；勿只改大 `pack.inp` 的 `number` 绕表单                |
| 无法上传 / 仓库无导入                      | 免费关闭；结构请画板生成                                        |
| 工作区配额不足                           | 免费 50 MB：删无用文件                                      |
| VIP 装盒给 `run_packmol.sh` / 超 10 万 | 减分子数或 [联系我们](https://www.molsimulx.com/contact_us/) |
| 测试模拟点不了                           | 缺 `data.lmp` / `in.test.lmp`，或超测试原子上限               |
| 任务红了 / 一直排队                       | 看任务日志；是否撞墙钟或他人占用并发                                  |
| 手改 `pack.inp` 后设置被盖掉              | 见 [资源管理器](M04-MDStudio资源管理器.md)                     |
| 界面异常、功能与文档不符、疑似 bug              | 记下 Tab、操作步骤与报错信息，经 [联系我们](https://www.molsimulx.com/contact_us/) 反馈；有效反馈按情况回馈积分 |

## 小结

1. 先分清轻重（与是否 VIP 无关）：**轻**含绘制、仓库、文件 IO、力场转换等；**重**为力场生成、装盒、测试模拟、超胞——主要限制在重量级。
2. **免费**可跑通孤立分子绘制 → 力场（mmff94 / am1-bcc）→ 装盒 → 测试（单分子 ≤10、装盒 ≤100、工作区 50 MB、无上传）。
3. **VIP** 更大原子与工作区，并开放周期分子、力场转换、超胞、上传与导入；机房仍有装盒约 10 万级硬顶。
4. 限额按**当前文件内容**统计，不以文件名为准。
5. 长跑与生产请下载后走集群；扩容与产品需求可 [联系我们](https://www.molsimulx.com/contact_us/) 反馈。
6. 遇到 bug、界面异常或文档与产品不一致，也欢迎 [联系我们](https://www.molsimulx.com/contact_us/) 反馈；有效 bug 反馈我们会按情况回馈积分。
