> **系列标签：** `MDStudio` · `限制` · `VIP` · `免费试跑`

MDStudio 能在浏览器里将分子结构变成 LAMMPS 输入，但它**不是**无限算力的集群节点：单分子太大、盒子装太满、测试模拟跑太久，都会被拦住——这是用来保护共享服务、也避免把「冒烟测试」当成生产仿真。

本文集中写**使用须知与限制**：免费试跑能做什么、VIP 多什么、规模与超时。数字以站点实际上线为准。逐步怎么点：[Quickstart：从画分子到测试模拟](M01-Quickstart从画分子到测试模拟.md)；界面地图：[MDStudio 功能与界面总览](M03-MDStudio功能与界面总览.md)。

## 一、先认清工具边界

| MDStudio 适合                           | 不适合                 |
| ------------------------------------- | ------------------- |
| 画/导入分子、生成力场、Packmol 装盒、写出 Lammps 输入文件 | 长时间生产 MD、GB 级轨迹在线存储 |
| 短时**测试模拟**验证输入能否启动、分子拓扑是否合理           | 取代课题组集群或本地并行作业      |
| 中小体系试错、教学演示、免费档跟练 Quickstart          | 绕过规模上限硬算超大盒子        |

生产轨迹、控温平衡、数据分析 → 下载输入后走本机 / 集群。平台搭建与长跑习惯见：[分子模拟工作平台搭建](../../在线资源/01-技术文档/T01-分子模拟工作平台搭建.md)、[本地与集群文件传输](../../在线资源/01-技术文档/T09-本地与集群文件传输.md)、[集群与 SLURM 简明教程](../../在线资源/01-技术文档/T10-集群与SLURM简明教程.md)。

当前在线规模与墙钟是**共享机房的保护线**。我们会根据用户使用反馈，逐步**扩容机器**或**对接高性能计算集群**，为更大体系和更长任务提供服务；额度与开放档位以届时官网与界面提示为准。需求可经官网 [联系我们](https://www.molsimulx.com/contact_us/) 反馈。

## 二、账号与权益

登录后按会员档位开放功能。**免费用户**可跟练 Quickstart 主流程；更大体系、更多工具与上传需 **VIP**。

### 2.1 功能开关

| 能力 | 免费 | VIP |
|------|------|-----|
| 登录、打开工作室 | ✅ | ✅ |
| **孤立分子**绘制 | ✅ | ✅ |
| **力场生成** | ✅（电荷见下） | ✅ |
| **搭建盒子**、**测试模拟** | ✅（规模见第三节） | ✅ |
| **分子仓库**浏览与 3D 预览 | ✅ | ✅ |
| 仓库**导入**到工作区 | ❌ | ✅ |
| 工作区**上传**本地文件 | ❌ | ✅ |
| 工作区文件管理（重命名、编辑、删除等） | ✅ | ✅ |
| **超胞 / 几何变换** | ❌（需 VIP） | ✅ |
| **周期分子**构建 | ❌（需 VIP） | ✅ |
| **力场转换**（LigParGen / ATB 等） | ❌（需 VIP） | ✅ |

点某 Tab 提示开通时，先确认登录态与会员状态，再查官网 [个人中心](https://www.molsimulx.com/user/) 与 [VIP](https://www.molsimulx.com/vip/)。

### 2.2 免费档电荷方法

力场生成里，免费仅可选：

- **mmff94**
- **am1-bcc**

**resp**、**eqeq**、以及部分高级选项需 VIP。Quickstart 用的 GAFF2 + AM1-BCC 在免费档可用。

### 2.3 建议怎么用免费档

按 [Quickstart](M01-Quickstart从画分子到测试模拟.md)：**画乙醇 → 力场 → 装盒约 10 个分子 → 测试模拟**。不要依赖上传或仓库导入；结构在孤立分子画板里生成即可。

## 三、规模与时间

限额分两层：**会员档位**（免费更紧）与**机房硬顶**（VIP 接近下表「VIP / 机房」列）。表单、SMILES、保存结构文件、以及手改过的 `pack.inp`，都会按当前内容重新统计后校验——**改文件名不能绕过**；改内容则按改后的原子数重新计量。

### 3.1 规模上限（体感数字）

| 类别         | 免费        | VIP / 机房（典型）          | 超出时你会看到                           |
| ---------- | --------- | --------------------- | --------------------------------- |
| **单分子原子数** | **≤ 10**  | 约 **1000**            | 绘制 / 力场 / 保存结构 / 装盒物种直接拒绝         |
| **装盒总原子**  | **≤ 100** | 在线 Packmol 约 **10 万** | 缩小分子个数；VIP 远超上限时可提示线下 / 定制或本地计算回传 |
| **测试模拟原子** | 受上装盒规模约束  | 约 **1000**            | 无法启动冒烟                            |
| **工作区总容量** | **50 MB** | 约 **200 MB**          | 提示空间不足，需清理文件                      |
| **上传单文件**  | 不可上传      | 约 **20 MB**           | 免费无上传入口；VIP 超限则失败                 |

> **Tips（免费）：** 乙醇约 9 原子，装盒 **10** 个 ≈ 90 原子，落在免费 100 原子装盒上限内。SMILES 也会先估算原子数；手改 `pack.inp` 里的 `number` 再点「运行 packmol」时，会按 `pack.inp` **重算**总原子。

> **Tips（VIP）：** 总原子略超在线上限时，界面仍可能给出 `run_packmol.sh` 供本地试跑后回传 `simbox.xyz`；**远超 10 万**请走官网 [联系我们](https://www.molsimulx.com/contact_us/) 线下定制。AM1-BCC / RESP 在原子数过阈值时，也可能改为「写脚本 + 挂起，上传计算结果再续跑」——按界面文案上传对应 `.out` 即可（需 VIP 上传权限）。

### 3.2 任务墙钟（两档共用量级）

| 任务 | 体感 | 超出时 |
|------|------|--------|
| 测试模拟 | 往往几十秒～数分钟 | 任务被终止，日志提示超时 |
| Packmol / 力场 | 可达数分钟 | 同上 |

具体秒数以界面 / 配置为准。

## 四、安全与习惯

- 勿向公共在线环境上传**未发表、涉密**数据。
- 大轨迹、完整生产输出放本地 / 集群存储，勿指望工作区当硬盘。
- 正式模拟：下载 `data.lmp`/`in.lmp` 后改到本机或集群。

## 五、翻车对照

| 现象                                 | 可先检查                                                      |
| ---------------------------------- | --------------------------------------------------------- |
| Tab 提示开通 VIP                       | 是否在「超胞 / 周期分子 / 力场转换」；绘制·力场·装盒·测试应对免费开放（见第二节）             |
| 电荷没有 resp / eqeq                   | 免费仅 mmff94、am1-bcc                                        |
| 力场 / 导出 / 保存报原子数超限                 | 免费单分子 ≤ **10**；缩小分子或开通 VIP                                |
| 装盒报总原子超免费上限                        | 减分子个数；勿不改表单却把 `pack.inp` 的 `number` 改大                    |
| 无法上传 / 仓库无导入按钮                     | 免费档关闭上传与导入；结构请画板生成                                        |
| 工作区提示配额不足                          | 免费 50 MB：删除无用文件后再试                                        |
| VIP 装盒提示 `run_packmol.sh` / 超 10 万 | 减分子数；或 [联系我们](https://www.molsimulx.com/contact_us/) 线下定制 |
| 测试模拟点不了                            | 是否缺 `data.lmp` / `in.test.lmp`；体系是否超冒烟上限                  |
| 任务红了 / 超时                          | 看任务日志；缩小体系或改参数后重试                                         |
| 手改 `pack.inp` 后设置被盖掉               | 见 [MDStudio 资源管理器（工作区文件）](M04-MDStudio资源管理器.md)：覆盖确认与备份习惯 |

## 小结

1. MDStudio 适合装输入与冒烟，不适合当无限集群。
2. **免费**可跑通绘制 → 力场（mmff94 / am1-bcc）→ 装盒 → 测试（单分子 ≤10、装盒 ≤100、工作区 50 MB、无上传）；跟练 [Quickstart](M01-Quickstart从画分子到测试模拟.md) 用乙醇 × 10。
3. **VIP** 放开超胞、周期分子、力场转换、上传与仓库导入，以及更大原子数与工作区；机房仍有 10 万级装盒硬顶。
4. 限额按**当前文件 / pack.inp 内容**统计，不以文件名为准。
5. 后续会根据使用反馈扩容或对接高性能集群，以官网 / 界面为准；可 [联系我们](https://www.molsimulx.com/contact_us/) 提需求。
