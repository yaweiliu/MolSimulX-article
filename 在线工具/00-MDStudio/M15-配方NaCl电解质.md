## 一、案例目标

| 项 | 建议 |
|----|------|
| **溶剂** | tip3p **或** tip4pew（二选一，全程一致） |
| **溶质** | Na+ / Cl−（库内同系列离子） |
| **目标** | 指定浓度直觉下的装盒输入 |
| **不算什么** | 电导率收敛、电极界面 |

---

## 二、点击路径（免费）

1. [分子仓库（浏览、预览与导入）](M08-分子仓库.md)：导入选定水模型 + 同系列 Na、Cl。
2. [搭建模拟盒子（Packmol 三步）](M11-搭建模拟盒子.md)：三（或更多）组分——水、钠、氯；数目满足电中性（Na 数 = Cl 数）。
3. Packmol → LAMMPS。
4. 可选冒烟：优先减少水分子数，保证总原子 < 测试上限。

---

## 三、文件名与参数含义

| 角色 | 说明 |
|------|------|
| 水 ff/结构 | 与所选 tip3p / tip4pew 条目一致 |
| 离子 | 勿混用「另一水模型配套离子」 |
| `pack.inp` | 水/Na/Cl 三个 number；盒长 |
| `data.lmp`/`in.lmp` | 生成输入 |

浓度：正文只强调中性与同系列；数值模板见会员包。

---

## 四、注意点

- tip3p 水 + tip4pew 离子 = 常见翻车。
- 高盐浓度时盒子与叠放更易 Packmol 失败 → 略放大盒子或减浓度再试。
