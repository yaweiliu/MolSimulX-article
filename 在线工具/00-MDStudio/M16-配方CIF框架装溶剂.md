MOF / 分子筛一类**框架 + 孔道溶剂/客体**，路径与溶液纯液不同：先有带胞的 CIF，力场与电荷选项也常走框架向。本篇给清单式点击路径；可跑 pack 与输入在会员包。

周期结构：[周期分子（晶体 / 表面 / 纳米结构）](M07-周期分子.md)；超胞：[超胞与几何变换](M10-超胞与几何变换.md)；力场：[力场生成（结构 / SMILES → .ff）](M09-力场生成.md)；限额：[MDStudio 使用须知与限制](M02-MDStudio使用须知与限制.md)。

## 一、案例目标

| 项 | 建议 |
|----|------|
| **框架** | 自备或周期分子生成的 `.cif`（原子数可控） |
| **客体** | 水或小分子溶剂 / 吸附质 |
| **目标** | 框架固定直觉下装入客体并写 LAMMPS |
| **不算什么** | 吸附等温线、柔性框架长时间动力学 |

---

## 二、点击路径（免费）

1. 准备 CIF：[周期分子（晶体 / 表面 / 纳米结构）](M07-周期分子.md) 或工作区上传自备 CIF。
2. 必要时 [超胞与几何变换](M10-超胞与几何变换.md) 小倍超胞（先算清原子数）。
3. [力场生成（结构 / SMILES → .ff）](M09-力场生成.md)：按界面选择框架适宜力场/电荷（如 EQeq 等体感选项；大体系可能续跑）。
4. 客体：仓库导入或自备力场（[分子仓库（浏览、预览与导入）](M08-分子仓库.md)/[力场生成（结构 / SMILES → .ff）](M09-力场生成.md)）。
5. [搭建模拟盒子（Packmol 三步）](M11-搭建模拟盒子.md)：框架 + 客体组分；注意 Packmol 对周期框架的界面选项（以 UI 为准）。
6. 生成输入；是否冒烟取决于总原子——框架一扩就很容易超测试上限。

---

## 三、CIF / 文件清单（免费）

| 清单项 | 说明 |
|--------|------|
| 源 `*.cif` | 有合理晶胞 |
| 变换后坐标 | 若做了超胞 |
| 框架 `.ff` + `_ff.xyz` | 力场产物 |
| 客体结构 + `.ff` | 溶剂/吸附质 |
| `pack.inp` / `simbox.xyz` / `data.lmp` / `in.lmp` | 装盒链 |

---

## 四、注意点

- RESP/EQeq 超阈值可能进入「脚本 + 回传」流程（须知）。
- 框架装盒失败时先检查胞是否过小、客体数目是否夸张。

## 小结

1. CIF 路径：结构 →（超胞）→ 框架力场 → 装客体。
2. 扩胞极易超限，先保守。
3. 成品输入见会员包。
