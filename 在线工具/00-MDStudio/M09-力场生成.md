> **系列标签：** `MDStudio` · `力场生成` · `GAFF`

有了 `.mol` / SMILES / CIF，还差「给原子贴上键合与电荷」——**力场生成**把结构变成装盒可用的 `{name}_ff.xyz` + `{name}.ff`。

本文讲：入口怎么选、GAFF 系与电荷场景、RESP/大分子续跑体感。限额见 [MDStudio 使用须知与限制](M02-MDStudio使用须知与限制.md)；结构来源见 [孤立分子（Ketcher 绘制）](M06-孤立分子Ketcher绘制.md) / [周期分子（晶体 / 表面 / 纳米结构）](M07-周期分子.md) / [分子仓库（浏览、预览与导入）](M08-分子仓库.md)。**不讲** 力场公式与参数拟合。

## 一、目标与前提

| 项 | 说明 |
|----|------|
| **目标** | 生成一对力场坐标与 `.ff` |
| **前提** | 工作区已有结构，或界面允许直接贴 SMILES；注意单分子原子上限 |
| **本 Tab 最易踩** | 只下载了 xyz 丢掉 `.ff`；或电荷方法选太慢导致超时 |

---

## 二、力场与电荷（入门场景表）

| 场景 | 力场倾向 | 电荷倾向 | 说明 |
|------|----------|----------|------|
| Quickstart / 小有机物 | GAFF2 | AM1-BCC | 快，适合冒烟 |
| 需要稍严谨一点 | GAFF / GAFF2 | RESP（若提供） | 更慢；可能触发续跑 |
| 周期性框架 | 界面给出的框架路径 | EQeq 等 | 与孤分子不同，跟 UI |

以当前站点选项为准；上表是写作时的常用组合。

---

## 三、操作步骤

1. 打开 **力场生成**。
2. 选择工作区结构，或提交 SMILES。
3. 选择力场与电荷方法。
4. 提交；等待完成或按提示上传中间结果续跑。
5. 刷新工作区，确认 `*_ff.xyz` 与 `.ff` 都在。

---

## 四、产物一览

| 产物 | 用途 |
|------|------|
| `{name}_ff.xyz` | 装盒用坐标 |
| `{name}.ff` | 参数；与上者成对 |

## 六、常见问题

| 问题 | 处理 |
|------|------|
| 原子数超限 | 缩小分子；对照须知 |
| 任务要你上传 `.out` | 按文案完成本地/脚本步骤再回传，属设计行为 |
| 装盒报缺参数 | 检查是否漏传或删了 `.ff` |

---

## 小结

1. 力场生成的核心产物是 `_ff.xyz` + `.ff` 一对。
2. 入门优先 GAFF2 + AM1-BCC。
3. 超时与续跑对照使用须知，不要盲目重点。
