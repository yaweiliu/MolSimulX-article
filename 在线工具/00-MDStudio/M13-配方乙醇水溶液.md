## 一、案例目标

| 项 | 建议 |
|----|------|
| **溶质** | 乙醇（绘制 / SMILES / 仓库，力场 GAFF2 + AM1-BCC） |
| **溶剂** | 仓库 **TIP3P** 水 |
| **目标** | 工作区得到含水体系的 `data.lmp` / `in.lmp`，可选冒烟 |
| **不算什么** | 精确摩尔浓度校验、长时间扩散采样 |

---

## 二、点击路径（免费）

1. **溶质**：按 [MDStudio Quickstart：从画分子到测试模拟](M01-Quickstart从画分子到测试模拟.md) 或 [孤立分子（Ketcher 绘制）](M06-孤立分子Ketcher绘制.md)→[力场生成（结构 / SMILES → .ff）](M09-力场生成.md) 得到乙醇 `_ff.xyz` + `.ff`。
2. **水**：[分子仓库（浏览、预览与导入）](M08-分子仓库.md) 检索 TIP3P → **导入**工作区（注意配套力场）。
3. **装盒**：[搭建模拟盒子（Packmol 三步）](M11-搭建模拟盒子.md) ① 添加乙醇与水两组分，定分子数与盒子 → ② Packmol → ③ 生成 LAMMPS。
4. **刚性水**：若界面有水模型 / shake 相关选项，按 TIP3P 常规保持刚性约束（以 UI 文案为准）。
5. **可选**：[测试模拟（LAMMPS 冒烟）](M12-测试模拟.md) 冒烟（总原子须低于测试上限；水分子数别一次加爆）。

## 三、文件名与参数含义（无完整脚本）

| 角色 | 你应看到的文件（举例） | 含义 |
|------|------------------------|------|
| 溶质坐标/力场 | `ethanol_ff.xyz`、`ethanol.ff` | 有机溶质 |
| 水 | 库导入的 tip3p 结构 + ff | 溶剂 |
| 装盒脚本 | `pack.inp` | 两组分数目与盒子 |
| 装盒结果 | `simbox.xyz` | 初始构型 |
| LAMMPS | `data.lmp`、`in.lmp` | 可下载输入 |

浓度相关：免费正文只强调「溶质个数 / 水个数 / 盒长」三者共同决定浓度；具体数字以会员包内 `pack.inp` 为准。

## 四、注意点

- 水与后续若加离子，保持**同系列**模型路径。
- 含水后原子数上升快，冒烟前先估总原子（[MDStudio 使用须知与限制](M02-MDStudio使用须知与限制.md)）。
- 正式平衡与分析 → 下载后离线；勿在线拉长跑。
