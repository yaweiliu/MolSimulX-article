> **系列标签：** `MDStudio` · `超胞` · `几何变换`

CIF 胞太小装不下溶剂、分子朝向不对、需要正交化一下——**超胞与几何变换** Tab 在装盒前把坐标摆顺，产物常见为 `*_trans.xyz`。

本文讲：扩胞 / 平移 / 旋转 / 正交化怎么用、何时做。限额见 [MDStudio 使用须知与限制](M02-MDStudio使用须知与限制.md)（变换后原子数暴涨会撞装盒与冒烟上限）。**不讲** 晶体对称群理论。

## 一、目标与前提

| 项 | 说明 |
|----|------|
| **目标** | 对已有结构做超胞或刚性几何变换 |
| **前提** | 工作区已有结构（xyz / cif 等，以界面支持为准） |
| **本 Tab 最易踩** | 扩胞倍数过大，后面 Packmol / 测试模拟全部超限 |

---

## 二、常见操作

| 操作 | 干什么 |
|------|--------|
| **超胞 / 扩胞** | 按 a、b、c 方向整数倍放大 |
| **平移** | 整体移动，便于装溶剂或对齐 |
| **旋转** | 调整分子/片层取向 |
| **正交化** | 把斜胞调到更易处理的坐标（若提供） |

---

## 三、操作步骤

1. 打开 **超胞变换**（界面标题以产品为准）。
2. 选择源结构。
3. 填写扩胞倍数或变换参数。
4. 生成；工作区出现 `*_trans.xyz`（或等价名）。
5. 后续力场 / 装盒请用**新文件**，不要混用旧坐标。

## 六、常见问题

| 问题 | 处理 |
|------|------|
| 变换后装盒提示过大 | 减小超胞；或走本地 `run_packmol.sh`（须知） |
| 找不到新文件 | 刷新工作区；检查命名 |
| 要不要先力场再超胞 | 看界面流程：有的路径先几何后力场；以 UI 与配方文为准 |

---

## 小结

1. 超胞 Tab 解决「太大/太小/朝向」几何问题。
2. 扩胞后原子数会乘上去，提前算冒烟与装盒上限。
3. 后续步骤统一使用变换后的文件。
