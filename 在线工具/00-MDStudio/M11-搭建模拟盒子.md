> **系列标签：** `MDStudio` · `Packmol` · `装盒`

结构和 `.ff` 都齐了，还差「放进周期盒子」——**搭建模拟盒子**把多组分塞进 Packmol，再写出 LAMMPS 的 `data.lmp` / `in.lmp`。这是 MDStudio 里步骤最多的一页。

本文按 **① 生成 pack.inp → ② 跑 Packmol → ③ 生成 LAMMPS** 写透。规模阈值与 `run_packmol.sh` 见 [MDStudio 使用须知与限制](M02-MDStudio使用须知与限制.md)；手改脚本进阶见 [配方：自定义 pack.inp（多组分 / 定密度）](M18-配方自定义pack.inp.md)。**不讲** Packmol 全选项手册。

## 一、目标与前提

| 项 | 说明 |
|----|------|
| **目标** | 得到可冒烟、可下载的 `data.lmp` + `in.lmp` |
| **前提** | 各组分已有坐标 + `.ff`（或仓库成套导入）；分子数落在在线阈值内 |
| **本 Tab 最易踩** | 跳步；或缺 `.ff`；或分子数过大触发本地下载脚本 |

---

## 二、三步流程

### ① 生成 `pack.inp`

1. 打开 **搭建盒子**，进入生成 pack 相关面板。
2. 添加组分：选择结构 / `.ff`；填分子数。
3. 设盒子边长或目标密度（按界面）。
4. 生成并写入工作区：`pack.inp`、各组分 pack 用坐标等。

多组分示例直觉：溶质 N 个 + 水 M 个；离子液体阴阳离子按化学计量比。

### ② 运行 Packmol

1. 确认 `pack.inp` 已在工作区。
2. 启动在线 Packmol。
3. 成功后得到 `simbox.xyz`。

若提示超过在线装盒上限：先缩小体系；需要 **10 万原子以上** 的盒子，见 [MDStudio 使用须知与限制](M02-MDStudio使用须知与限制.md)（可 [联系我们](https://www.molsimulx.com/contact_us/) 线下定制）。略超阈值时也可能下载 `run_packmol.sh` 本机装完再上传 `simbox.xyz`（[MDStudio 资源管理器（工作区文件）](M04-MDStudio资源管理器.md)）。

### ③ 生成 LAMMPS 输入

1. 确认 `simbox.xyz` 与力场文件齐套。
2. 生成 `data.lmp`、`in.lmp`。
3. 下载备份到本机（好习惯）。

## 五、常见问题

| 问题 | 处理 |
|------|------|
| Packmol 失败 | 读日志：盒子是否过小、分子重叠、文件路径 |
| 生成 LAMMPS 缺力场 | 检查每个组分 `.ff` 是否在工作区 |
| 改完 pack 又被重置 | 见资源管理器覆盖说明 |

---

## 小结

1. 装盒固定记三步：pack.inp → Packmol → data/in.lmp。
2. 过大体系会改走本地下载脚本，属限额而非故障。
3. 冒烟与生产长跑分开：生产请下载后离线跑。
