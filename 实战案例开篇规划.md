## 三、开篇文章清单（建议先写这 6 篇）

> 目标：用 **6 篇** 覆盖「体相热力学 → 受限 → 控压技巧 → 轨迹分析 → 粗粒化混合」，全部绑定 Archive / 研究笔记，形成可引流的第一条实战路径。

### C10 · CO₂ 分子动力学与状态方程

| 项 | 内容 |
|----|------|
| **科学问题** | 用 MD 得到 CO₂ 的 P–V（或 ρ–P）关系，理解压缩/膨胀路径与 EOS |
| **素材** | `example_CO2_EOS`（`co2_eos.ipynb`、`in.lmp`、`input_compression/expansion.lmp`、结果 log/轨迹片段） |
| **流程骨架** | zmat/ff → fftool+packmol 装盒 → LAMMPS 压缩/膨胀 → 读 thermo → 画 EOS / 对比实验或经验式（点到为止） |
| **前置** | T01；Packmol/fftool 速成（或案例内附最小步骤）；K02/K11/K16 |
| **交付** | Notebook + 精简输入；可选短轨迹包（download） |
| **paywall 倾向** | 导语+装盒免费钩子；生产段与完整分析 VIP / 包 |
| **笔记呼应** | 与「简单液体」教学同级；比氩气 LJ 更贴近 Archive 现成数据 |

### C11 · 简单液体：自扩散与粘度（LAMMPS）

| 项 | 内容 |
|----|------|
| **科学问题** | 从平衡轨迹算自扩散；从应力相关或粘度公式得动力粘度（选一种主路径写透） |
| **素材** | 研究笔记 2021/03/28；可用 LJ 流体或延续 C10 的 CO₂/简易液体盒子（若笔记输入可找回则优先） |
| **流程骨架** | 平衡 NVT/NPT → 生产 → MSD→D；粘度算法与有限尺寸注意（链 K18/K21） |
| **前置** | C10 或等价体相跑通；MDAnalysis 速成；K16、K21 |
| **交付** | 分析 Notebook + 示例 log/短轨迹 |
| **说明** | 手册旧草案「氩气 LJ NVT」可并入本篇或作为本篇的 LJ 附录体系 |

### C12 · 两墙夹水：受限液体建模与开跑

| 项 | 内容 |
|----|------|
| **科学问题** | 如何在两固体壁面之间放入水并生成可跑的 LAMMPS 输入；看密度剖面/结构（入门级） |
| **素材** | `example_water_between_two_walls`（`water_between_walls.ipynb`、spce、wall、pack） |

| **paywall 倾向** | 建模与输入生成可作免费门面；完整分析 VIP |

### C13 · 机械式控压（Mechanical barostat）

| 项 | 内容 |
|----|------|
| **科学问题** | 不用常规压浴时，如何用「机械」方式调控压力；看 thermo / 应力相关输出 |
| **素材** | `example_mechanical_barostat` + 研究笔记 2021/03/08 |
| **流程骨架** | 读懂 in 与活塞/墙设置 → 跑短算例 → 解析 `result_*` → 与 NPT 压浴对比（概念，链 K11） |

### C14 · 离子液体：阳离子质心与取向分析

| 项 | 内容 |
|----|------|

### C15 · 粗粒化：BmimPF₆–乙腈混合液

## 六、每篇统一结构（写作清单）

1. **场景钩子**（要算什么、和实验/论文哪类问题有关）  
2. **你将得到**（可跑通的输入 + 一张主图 + 可引用的分析步骤）  
3. **前置**（环境、工具速成、相关 K）  
4. **体系与力场**（文件从哪来、为何这样选）  
5. **建模与输入**（packmol/fftool/ASE 关键步骤）  
6. **运行**（LAMMPS 节选；集群点到为止，链 T10）  
7. **分析与图**（Notebook；块平均/误差能提则提，链 K17）  
8. **常见翻车**  
9. **资源包说明** + **学习路径**  

资源包建议最小集：`in`/`data` 或生成脚本、短轨迹或 thermo log、画图 Notebook；大轨迹外置下载。
